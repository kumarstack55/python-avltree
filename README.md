# python-avltree

## 利用上の要件

* PyPy3.7 または Python3.9以降

## 開発上の要件

* poetry

## テストする

```
# 1つのPythonについてテストする
poetry run pytest

# すべてのPythonについてテストする
poetry run tox
```
